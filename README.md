gedit-sudo-reopen
============

A plugin for gedit that prompts for the root password and re-opens the current document as root.

* [Website](http://github.com/elijahr/gedit-sudo-reopen)


Installation
------------

    $ cp sudo_reopen* ~/.gnome2/gedit/plugins
